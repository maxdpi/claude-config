| `PLATFORM-ASSUMPTIONS.md` | M-000 validated platform assumptions (A1-A4) with confidence and proofs | Reviewing substrate design decisions, implementing resume or hooks   |
| `update_official_claude_docs.py` | uv script that mirrors every Claude Code doc listed in `code.claude.com/docs/llms.txt`, downloading each page as raw markdown (no API key), sorting it into a section folder that mirrors the docs sidebar, and regenerating `README.md` + `CLAUDE.md` for the tree. Defaults to a `claude-code/` folder in the cwd; pass `-d <dir>` to target another location, `--filter <substr>` to limit pages. | Refreshing the vendored Claude Code docs tree |
